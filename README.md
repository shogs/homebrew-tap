# homebrew-tap

Homebrew tap for installing git-wt and other tools.

## Installation

First, tap this repository:

```bash
brew tap shogs/tap
```

Then install git-wt:

```bash
brew install git-wt
```

Or install directly in one command:

```bash
brew install shogs/tap/git-wt
```

## Available Formulae

- **git-wt**: A comprehensive Git worktree management CLI tool
